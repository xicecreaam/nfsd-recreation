# Setting up for speedruns

All the information needed to set up the PC version of Most Wanted for Least Wanted speedruns.

<span style={{ fontSize: '2rem', fontWeight: 'bold' }}>To setup the game properly and correctly, see [this](https://xicecreaam.github.io/nfsd-recreation/mw05/any/setup/).</span>

***

## Differences with vanilla game

* You can use any language you want. Available languages are : Danish, Dutch, English, Finnish, French, German, Italian, Polish, Spanish and Swedish.  
> Regarding certain languages (Danish, Dutch, Finnish, Polish and Swedish) don't have dubbed audio. It will use English speeches except for certain ones. See [this](https://www.youtube.com/watch?v=iz4xqcg97OY) to notice what changes.
> Other languages (Russian and Asian ones) are incompatible because they are either **1.3 forced** or **lap glitch doesn't work**.  
* This setup is for PC only since it needs to be played on a PC version.  
* You need a **Black Edition copy** to get additional content. If you don't have the `INGAMEC.BUN` file and would like to enable Black Edition content, you can [download it from speedrun.com](https://www.speedrun.com/nfsmw05/resources/cngqz).  
* Disable the `Speech` audio volume in the Settings menu by setting the cursor to `0`. This removes cop speeches completely and prevents a cop spawn during a race, resulting in a time loss.
